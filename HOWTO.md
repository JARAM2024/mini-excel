# How to build

## Windows
### Caution
- Hot compiling is not available now.

clang.exe -g3 main.c -o mini-excel

## MacOS

clang -g3 main.c -o mini-excel

# How to run

## Windows
### Caution
- Hot compiling is not available now.

./mini-excel

## MacOS

CC="\<path of your compiler\>" ./mini-excel
